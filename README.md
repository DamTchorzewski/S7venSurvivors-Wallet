### Dependencies

The LTS version of [Node. js](https://nodejs.org/en/) must be installed on the computer.

### Overruled with Vite

1. Clone repo
2. Create your branch
3. Install Resources:

```shell
npm install
```

4. Enable working mode:

```shell
npm run dev
```

In your browser, go to [http://localhost:1234](http://localhost:1234).
